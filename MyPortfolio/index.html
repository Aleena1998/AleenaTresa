Gynachology Hospital Management System

Helmet Violation System

The Helmet Violation System is a web and mobile application developed using Python Flask, MySQL, and Android Studio. Additionally, advanced technologies like CNN (Convolutional Neural Network) are employed for number plate detection. The system was primarily developed to assist Motor Vehicle Departments (MVD) in enforcing helmet regulations. It operates by capturing images of motor vehicles on the road and checking whether the rider is wearing a helmet or not. If a rider is found without a helmet, the system checks the number plate and issues a fine. Users receive proof of the violation and can conveniently make online payments for the fine.
